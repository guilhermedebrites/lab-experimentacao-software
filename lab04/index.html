RQ1
As linguagens mais utilizadas variam conforme o modelo de trabalho?
Selecionar modelos:
Linguagens × Modelo de Trabalho (100%)
Proporção de cada modelo de trabalho por linguagem
Popularidade × Trabalho Remoto por Linguagem
Cada ponto é uma linguagem. Tamanho = nº de usuários. Eixo X = % remoto.
💡 Insight: A distribuição entre modelos de trabalho é notavelmente homogênea em todas as linguagens analisadas. TypeScript lidera em trabalho remoto (36,3%), seguida de JavaScript (34,8%). No outro extremo, C++ (25,0%) e Java (28,4%) concentram proporcionalmente mais trabalho presencial — possivelmente pela natureza dos setores que as utilizam, como indústria, sistemas embarcados e finanças.

Cada caixa representa a distribuição de anos de experiência dos devs naquele grupo de confiança
💡 Insight: Surpreendentemente, a experiência profissional não determina maior confiança na IA. A proporção de desconfiança é consistente em todas as faixas (~40-45%), sugerindo que o ceticismo em relação à IA é transversal à senioridade — tanto iniciantes quanto desenvolvedores sênior com 20+ anos demonstram padrões similares de desconfiança.
